# 测试验收清单

## 1. 说明

本清单用于 MVP 验收，不用于完整工业级测试体系。

验收重点：

- 主流程是否跑通
- 模拟平台是否可用
- 单平台 PoC 是否可演示
- AI 是否只建议不自动发
- 失败是否能安全降级
- 是否存在误发风险
