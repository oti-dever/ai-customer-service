## 一、文档目的
定义界面风格、布局规则、组件样式、设计 Token。供 UI 开发与 AI 辅助生成时参照，保证视觉与交互一致。规范基于轻简聚合客服等参考 UI 截图提炼。

---

## 二、技术栈
+ **Qt Widgets**（唯一 UI 框架）
+ **QSS**（Qt Style Sheets）用于样式

现阶段仅使用 Qt Widgets + QSS，不使用 QML。下述规范均为可在该技术栈下实现的效果。

---

## 三、设计 Token
Token 为给颜色、间距、字体、圆角等设计参数起的名字，设计 Token 是把设计中反复出现的视觉参数，用命名变量固定下来，而不是到处写具体数值，在一个地方集中定义，方便复用和修改，保证界面风格统一。

### 3.1 颜色
| Token | 用途 | 值（示例） |
| --- | --- | --- |
| primary | 主色（按钮、链接、选中态） | #1890FF |
| primary_hover | 主色悬停 | #40a9ff |
| accent_orange | 强调色（聚合接待选中、待处理等） | #FF8C42 / #FA8C16 |
| accent_green | 成功/启动/保存 | #52c41a |
| accent_purple | 辅助强调（机器人、对话相关） | #722ED1 |
| accent_red | 危险/AI 配置/错误 | #ff4d4f |
| success | 成功 | #52c41a |
| warning | 警告 | #faad14 |
| error | 错误 | #ff4d4f |
| bg_main | 主背景 | #F5F5F5 / #F0F2F5 |
| bg_card | 卡片背景 | #FFFFFF |
| bg_sidebar | 侧边栏（深蓝渐变） | #001529 → #1a2a4a |
| bg_banner | 信息横幅 | #E6F7FF |
| text_primary | 主文字 | #262626 |
| text_secondary | 次要文字 | #8C8C8C |
| text_on_dark | 深色背景上的文字 | #FFFFFF / rgba(255,255,255,0.9) |
| border | 边框 | #E8E8E8 |


### 3.2 间距
| Token | 值 | 用途 |
| --- | --- | --- |
| space_xs | 4px | 最小间距 |
| space_sm | 8px | 小间距 |
| space_md | 16px | 中等间距 |
| space_lg | 24px | 大间距 |
| space_xl | 32px | 超大间距 |


### 3.3 圆角
| Token | 值 | 用途 |
| --- | --- | --- |
| radius_sm | 4px | 小控件、标签 |
| radius_md | 8px | 输入框、按钮、卡片 |
| radius_lg | 12px | 大卡片、登录面板 |


### 3.4 字体
| Token | 值 | 用途 |
| --- | --- | --- |
| font_family | 微软雅黑 / PingFang SC | 默认 |
| font_size_sm | 12px | 辅助文字、标签 |
| font_size_base | 14px | 正文 |
| font_size_lg | 16px | 标题 |
| font_size_xl | 20px | 大标题、欢迎语 |


### 3.5 阴影（QSS 实现方式）
QSS 不支持 `box-shadow`，可采用以下方式模拟层次感：

+ **边框模拟**：`border: 1px solid rgba(0,0,0,0.06)` 或略深边框色
+ **背景色差**：卡片略浅于主背景，形成对比
+ **可选**：需真实投影时，可在代码中为 `QFrame` 等使用 `QGraphicsDropShadowEffect`，而非 QSS

### 3.6 登录页背景与卡片（Qt Widgets + QSS 方案）
QSS 不支持 `backdrop-filter`（磨砂 blur），故采用**纯色 + 渐变**实现：

+ **背景**：`qlineargradient` 紫色渐变（#7C3AED → #A78BFA）
+ **登录卡片**：白色或浅色背景（`#FFFFFF` / `#F8F5FF`），圆角、边框，居中
+ **输入框**：白底、边框、圆角，占位符浅灰

## 五、组件规范
### 5.1 按钮
| 类型 | 样式 | 典型场景 |
| --- | --- | --- |
| 主要 | 主色/绿色/黑色背景，白字 | 保存、创建、立即登录 |
| 次要 | 白底，主色边框或文字 | 重置、取消 |
| 彩色 | 橙/紫/蓝/绿等背景，白字 | 管理行业、管理平台、AI 配置 |
| 危险 | 红色背景或边框 | 删除、警告操作 |
| 文字 | 无背景，主色文字 | 注册账号、忘记密码 |


+ 带图标按钮：图标 + 文字，间距约 8px

### 5.2 输入框
+ 高度约 32px，圆角 `radius_md`
+ 边框 1px，#E8E8E8
+ 聚焦时主色边框
+ 可带左侧图标（如人形、锁、放大镜）、占位符提示

### 5.3 搜索框
+ 左侧放大镜图标，占位符如「Q 搜索功能...」「搜索会话或客户名...」
+ 可与蓝色「搜索」按钮组合

### 5.4 开关（Toggle）
+ 开启：主色（蓝）背景
+ 关闭：灰色
+ 右侧可配说明文案

### 5.5 表格
+ 表头背景略深
+ 行悬停高亮
+ 支持多选时左侧复选框

### 5.6 卡片
+ 白色背景，圆角，边框或背景色差体现层次（QSS 不支持 box-shadow）
+ 可带右上角三点菜单（`QToolButton` 等）
+ 数据概览卡片可用颜色主题（蓝/绿/紫/橙）区分

### 5.7 空状态
+ 居中大图标 + 主标题 + 副标题 + 可选主操作按钮
+ 示例：
+ 「暂无会话」+「选择会话开始聊天」
+ 「还没有机器人」+「创建您的第一个 AI 客服机器人」
+ 「选择节点开始管理」+「从左侧树状导航中选择一个节点」

### 5.8 信息横幅（Banner）
+ 浅蓝背景 `#E6F7FF`，左侧勾选/提示图标
+ 可关闭（右侧 x）
+ 用于兼容性提示、成功/警告/错误信息

### 5.9 标签页（Tabs）
+ 选中项：主色下划线或背景
+ 如「基础设置」「商品说明书」等

### 5.10 树形结构
+ 展开/收起箭头，节点前图标
+ 右侧可显示数量；支持加号新增子节点

## 六、导航与图标
### 6.1 左侧导航
+ 顶部：应用名 + 图标
+ 今日概况：进度展示（可用 `QProgressBar` 横向或自定义 `QWidget` 绘制圆环）、算力剩余、购买额度等
+ 核心功能 / 对话管理 / 系统管理：分组标题 + 菜单项（`QListWidget`、`QTreeWidget` 等）
+ 每项：左侧图标 + 主标题 + 可选副标题
+ 选中态：深色背景块 + 白色文字，或左侧竖条强调

### 6.2 图标
+ 风格：简洁线框或实心，扁平化
+ 常用：火箭、机器人、书本、聊天气泡、齿轮、扳手、闪电、禁止、循环箭头等
+ 与文字对齐、统一尺寸（如 16–20px）

## 七、交互规范
### 7.1 加载
+ 列表加载：`QProgressBar`  indeterminate 模式或居中加载文案（骨架屏需自定义 `QWidget`，可简化）
+ 按钮提交：`setEnabled(false)` + 禁用态样式

### 7.2 反馈
+ 成功：Toast 或状态栏提示
+ 错误：弹窗或行内红色提示（如「请先输入令牌」）
+ 确认操作：二次确认（如删除）
